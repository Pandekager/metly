# Phase 26 Plan: Insights Dashboard

## Objective
Create a unified Insights Dashboard that aggregates insights from all previous analysis phases (20-25) into actionable recommendations.

## Context
Previous phases completed:
- Phase 20: Order Flow Analysis
- Phase 21: Revenue Leak Detection
- Phase 22: Refund & Return Analysis
- Phase 23: Checkout Drop-off Analysis
- Phase 24: Operational Bottlenecks
- Phase 25: Customer Behavior Insights

## Tasks

### Task 1: Backend - Create insights_dashboard.py endpoint
**Type:** auto
**Files:** backend/src/endpoints/insights_dashboard.py

Create the aggregation endpoint that:
- Queries data from all analysis endpoints
- Aggregates top revenue leak sources
- Identifies top bottlenecks
- Lists top refunded products
- Calculates key retention metrics
- Generates actionable recommendations with priority scores
- Returns structured response with all insights

### Task 2: Frontend - Create InsightsDashboard.vue component
**Type:** auto
**Files:** frontend/components/charts/InsightsDashboard.vue

Create the dashboard component that:
- Displays top revenue leak sources in a card/list
- Shows top bottlenecks with severity indicators
- Lists top refunded products with refund rates
- Shows key retention metrics (LTV, retention rate)
- Displays actionable recommendations with priority badges
- Uses Tailwind CSS with proper dark mode support
- Follows existing component patterns

### Task 3: Frontend - Add Indsigt tab to home.vue
**Type:** auto
**Files:** frontend/pages/home.vue

Add the "Indsigt" tab that:
- Adds tab entry to tabs array
- Conditionally renders InsightsDashboard component
- Adds import for InsightsDashboard component
- Lazy loads data when tab is selected

## Verification
1. Backend endpoint returns aggregated insights
2. Frontend dashboard displays all insight categories
3. Tab navigation works correctly
4. Dark mode support functional

## Output Specification
- Backend: `backend/src/endpoints/insights_dashboard.py`
- Frontend: `frontend/components/charts/InsightsDashboard.vue`
